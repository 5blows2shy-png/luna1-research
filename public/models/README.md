# Luna1 Valuation Models

Place only reviewed, publication-ready Excel models in this directory.

Naming convention: `TICKER-Luna1-Valuation-Model.xlsx`

After adding a model, update the corresponding `model` record in
`src/data/research/research-companies.ts`:

- set `status` to `available`;
- set `url` to `/models/TICKER-Luna1-Valuation-Model.xlsx`;
- add the verified file size and model version;
- replace `Date to be confirmed` with the verified revision date.

Do not publish blank templates or models containing unsourced figures.
